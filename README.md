 Azure AI Foundry Agent

A beginner-friendly conversational AI agent built using Azure AI Foundry and TypeScript.

 Overview

This project demonstrates how to create, configure, and interact with an AI agent using Microsoft's Azure AI Foundry Agent Service. The agent can maintain conversations, process user requests, and serve as a foundation for more advanced AI applications.

 Features

* Conversational AI interactions
* Azure AI Foundry integration
* TypeScript implementation
* Secure credential management using environment variables
* Simple and extensible architecture for future enhancements

Tech Stack

* TypeScript
* Node.js
* Azure AI Foundry
* Azure Identity SDK

 Future Improvements

* Custom tool integration
* Document search and retrieval (RAG)
* Web search capabilities
* Multi-agent workflows
* User interface dashboard
